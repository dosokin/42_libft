# Libft

Custom implementation of a subset of the C standard library, developed as part of the 42 School curriculum.

## Overview

Libft consists of recreating common functions from the C standard library as well as additional utility functions.
The project focuses on understanding low-level programming concepts such as memory management, pointers, and data structures.

## Tech Stack

* C
* Makefile
* GCC

## Features

* String manipulation functions (`ft_strlen`, `ft_strdup`, `ft_strjoin`, etc.)
* Memory management functions (`ft_memset`, `ft_memcpy`, `ft_calloc`, etc.)
* Character checks and conversions (`ft_isalpha`, `ft_isdigit`, `ft_tolower`, etc.)
* File descriptor output functions (`ft_putchar_fd`, `ft_putstr_fd`, etc.)
* Linked list utilities (`ft_lstnew`, `ft_lstadd_back`, `ft_lstclear`, etc.)

## Learning Outcomes

* Manual memory management
* Pointer manipulation
* Understanding of core C standard library functions
* Building reusable static libraries
